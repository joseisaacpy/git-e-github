Principais Comandos e Conceitos

  • git init

    Inicializar um repositório Git.

  • git status

    Verificar o status do repositório.

  • git add .

    Adicionar todos os arquivos para o stage, preparando-os para serem commitados.

  • git add `nome do arquivo`

    Adicionar um único arquivo para o stage, preparando-o para ser commitado.

  • git commit -m `mensagem`

    Commitar os arquivos do stage com uma mensagem explicativa.

  • git config --global user.email "seuemail@example.com"

    Definir o e-mail globalmente para que o Git não peça novamente.

  • git config --global user.name "Seu Nome"

    Definir o nome de usuário globalmente no Git.
